# Thesis Scraper

## 概要

本プログラムは、作成者が所属する大学の卒業研究一覧ページを巡回し、各学生の卒業研究PDFから研究タイトルを自動抽出してCSVファイルへ保存するPythonプログラムです。

HTMLスクレイピングとPDF解析を組み合わせることで、卒業研究一覧を効率的に収集できます。

---

# 主な機能

* 卒業研究一覧ページの取得
* 学生ごとのPDFリンクの収集
* PDFファイルの自動ダウンロード
* PDF1ページ目から研究タイトルを抽出
* 不要な文字列の除去
* CSVファイルへの保存
* テストモードによる少数件の実行

---

# 使用ライブラリ

* requests
* BeautifulSoup4
* urllib3
* pdfplumber

インストール例

```bash
pip install requests beautifulsoup4 pdfplumber urllib3
```

# 対象サイト

```text
非公開
```

プログラムはこの一覧ページから各グループページへ移動し、卒業研究PDFを取得します。

---

# 設定

プログラム冒頭の設定項目を変更することで動作を制御できます。

| 変数            | 内容          |
| ------------- | ----------- |
| BASE_LIST_URL | 卒業研究一覧ページ   |
| TEST_MODE     | テスト実行の有無    |
| TEST_LIMIT    | テスト時に処理する件数 |
| OUTPUT_CSV    | 出力CSVファイル名  |

例

```python
TEST_MODE = True
TEST_LIMIT = 5
```

とすると最初の5件のみ処理します。

---

# 処理の流れ

## 1. 一覧ページの取得

一覧ページへアクセスし、

* グループページ

へのリンクを取得します。

---

## 2. PDFリンク収集

各グループページから

* 学生氏名
* PDFファイルURL

を取得します。

---

## 3. PDF解析

各PDFについて

* ダウンロード
* 1ページ目を取得
* テキスト抽出

を行います。

PDF解析には **pdfplumber** を使用しています。

---

## 4. タイトル抽出

1ページ目上部の文字列から正規表現を利用して研究タイトルのみを抽出します。

抽出時には以下のような不要情報を除去します。

* 学籍番号
* 氏名
* ページ番号
* 年度
* 「卒業研究」などの固定文言

例

```text
非公開
```

↓

```text
非公開
```

---

## 5. CSV出力

取得した情報をCSVとして保存します。

出力項目

| 列名      | 内容              |
| ------- | --------------- |
| 学籍番号・氏名 | 一覧ページの表示名       |
| PDF_URL | PDFのURL         |
| 抽出タイトル  | PDFから抽出した研究タイトル |

---

# 出力ファイル

```
thesis_list_cleaned.csv
```

CSVはUTF-8（BOM付き）で保存されるため、Microsoft Excelでも文字化けしにくい形式となっています。

---

# エラー処理

以下の場合にはエラーメッセージを返します。

* Webサイトへアクセスできない
* PDFダウンロード失敗
* PDF解析エラー
* テキスト抽出不可

処理は可能な限り継続されます。

# 注意事項

* 対象Webサイトの利用規約やアクセスルールを遵守してください。
* サーバー負荷を軽減するため、プログラムではアクセス間隔を設けています（`time.sleep()`）。
* WebサイトのHTML構造やPDFレイアウトが変更された場合は、スクレイピング処理やタイトル抽出ロジックの修正が必要になる場合があります。
